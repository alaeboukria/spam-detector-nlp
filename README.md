# 🛡️ Spam & Threat Detector (NLP)

*Projet réalisé dans le cadre d'un stage technique au CHU de Fès.*

## 📝 Description du projet
Ce projet est un système d'analyse textuelle conçu pour identifier et filtrer les menaces telles que le Phishing et les Spams. L'objectif était de renforcer la logique d'identification des anomalies au sein d'un flux de données en utilisant des techniques de traitement du langage naturel (NLP).

## 🛠️ Technologies Utilisées
* **Langage :** Python
* **Machine Learning & NLP :** NLTK, Scikit-learn, Pandas (à adapter selon ce que tu as vraiment utilisé)
* **Concepts :** Classification de texte, Tokenization, Analyse de fréquence

## ⚙️ Fonctionnalités Principales
1. Prétraitement et nettoyage des données textuelles.
2. Entraînement d'un modèle de classification pour différencier les messages légitimes des menaces.
3. Qualification des requêtes suspectes avec un score de probabilité.
